== Description ==

This free plugin connects myCred with Events Manager Pro. While myCred has built-in support for the free version, this plugin is required as of version 1.7.9.1 to support the PRO version.

= Built-in Support =
Here are the most notable features of myCred for Events Manager Pro plugin.

* Allow users to pay for tickets using points.
* Setup your event booking to only accept points as payment.
* Supports Multibookings.
* Supports Manual Approvals of bookings.
* If users can cancel bookings, you can select to refund their point payment.
* Supports profit sharing with event owners.

= Plugin Requirements =

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)

* WordPress 5.0+
* PHP 5.3+

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for Events Manager Pro
3. Find myCred for Events Manager Pro and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
